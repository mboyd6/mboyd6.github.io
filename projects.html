RGB Color Guessing Game

This is a simple game that helps web designers learn their colors in rgb(x, y, z) format! I used to be dependent on a color picker tool when I was trying to find that perfect color. After playing this color game for 30 minutes, I don't need to use a color picker anymore!

I made this color guesser to get used to manipulating the DOM with vanilla javascript. This project also helped me get my feet wet with jQuery, and solidified my confidence with CSS.
